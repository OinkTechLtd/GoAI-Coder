# Tips and Best Practices

To get the best results from the AI engineer at [coder-goai.lovable.app](https://coder-goai.lovable.app):

- **Be Specific**: Mention specific libraries if you prefer them (e.g., 'Use Lucide icons').
- **Error Handling**: If you see a bug in the preview, copy the error message and paste it into the chat.
- **Responsive Design**: Always check how your app looks on mobile by resizing the preview window.
- **Large Projects**: Break large applications into smaller tasks. Prompt for one page at a time.
