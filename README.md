# Coder-GoAI

Welcome to Coder-GoAI – your AI-driven coding assistant. This platform helps you transform ideas into functional applications using natural language.

## 🚀 Quick Start

1. Visit [coder-goai.lovable.app](https://coder-goai.lovable.app)
2. Enter your project prompt in the main input field.
3. Watch as the AI generates code, previewing the results in real-time.
4. Refine your app by chatting with the assistant to add features or fix bugs.

## 📖 Documentation

Detailed guides can be found in the `/docs` directory:
- [Getting Started](./docs/getting-started.md)
- [Best Practices](./docs/tips.md)

## 🛠 Features

- **Natural Language Prompting**: Describe what you want in plain English or Russian.
- **Live Preview**: See changes instantly in the integrated browser.
- **Version History**: Revert to previous versions if needed.
- **Seamless Export**: Deploy or download your generated code.
